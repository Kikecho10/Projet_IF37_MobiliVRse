Ceci devait être l'application pour visionner les vidéos via l'occulus. Il n'y finalement que les assets (sans vidéos) pour que le fichier ne soit pas trop lourd pour git hub.

Guide d'utilisation de l'application :

La video de base est une prise de vue en 180 d'un appartement fait à ~1m50 du sol (test avant le robot)

Clavier :

Appuyez sur la touche A pour lancer la video C:/Videos_360/A.MP4
et ainsi de suite pour les touches A, Z, E, R, T, Y et U.

Appuyer sur C pour mettre pause et sur V pour play.
Vous pouvez aussi maintenir la touche espace.

Appuyez sur M pour désactiver la video (écran noir) et sur L pour la remettre.

Occulus : 

Gachettes : play/pause

Boutons (A et B) : désactiver la video (écran noir) et la remettre
